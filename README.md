## 🎯 Concept

China Square est une boutique e-commerce moderne axée sur les **prix les plus bas** du marché dropshipping. Notre modèle :
- Prix AliExpress + **$2 markup fixe**
- Large catalogue de produits pas chers
- Design ultra-moderne style marketplace

## 📦 Dropshipping avec Shopify

Pour une solution **100% automatisée** (recommandé) :

### 1. Créer compte Shopify
- Allez sur [shopify.com](https://shopify.com)
- Démarrez votre essai gratuit 14 jours
- Choisissez un nom de boutique

### 2. Installer DSers (gratuit)
- Depuis Shopify App Store
- Connecter votre compte AliExpress
- DSers synchronise automatiquement les produits

### 3. Configurer le markup automatique
Dans DSers :
```
Prix vente = Prix AliExpress + $2
```

### 4. Importer les produits
- Rechercher sur AliExpress via DSers
- Filtrer par prix < $5
- Importer en 1 clic
- Les prix se mettent à jour automatiquement

### 5. Personnaliser le thème
- Importez le design China Square (rouge/jaune)
- Ou utilisez un thème Shopify existant et modifiez les couleurs
